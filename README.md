A plain and simple slideshow, based on [React](https://facebook.github.io/react) and LocalStorage

## Requirements
- NodeJS
- A browser with the [HTML5 localStorage API](http://www.w3.org/TR/webstorage/#the-localstorage-attribute)

## Usage

```sh
$ npm install
```

```sh
$ gulp build
```

```sh
$ gulp serve
```
